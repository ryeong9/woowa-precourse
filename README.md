# javascript-racingcar-precourse

### ✅ 기능

**1. 입력**

- 경주할 자동차 이름을 쉼표(,)로 구분하여 입력받는다.
- 각 이름은 5자 이하만 허용한다.
- 이름이 6자 이상일 경우 `[ERROR] 자동차 이름은 5자 이하만 가능합니다.` 예외 발생.
- 쉼표(,)가 아닌, 다른 특수문자가 포함되는 경우 `[ERROR] 자동차 이름은 쉼표(,)로만 구분해야 합니다.` 예외 발생.
- 빈 문자열의 경우 `[ERROR] 자동차 이름 사이에 빈 값이 있습니다.`예외 발생.
- 이름이 중복되는 경우 `[ERROR] 자동차 이름은 중복될 수 없습니다.` 예외 발생.
  <br>
- 시도할 횟수를 입력받는다.
- 입력한 횟수가 숫자가 아닐 경우 `[ERROR] 시도 횟수는 숫자여야 합니다.` 예외 발생.
- 음수일 경우 `[ERROR] 유효하지 않은 시도 횟수입니다.` 예외 발생.

**2. 실행**

- 각 자동차는 매 시도마다 0~9 사이의 무작위 값을 뽑는다.
- 무작위 값이 4 이상이면 전진, 아니면 멈춘다.
- 전진 하면 하이폰(-) 하나 추가하여 표현한다.
- 시도 횟수만큼 반복하며 각 턴마다 자동차들의 이동 결과를 출력한다.
  ```
  pobi : --
  woni :
  jun : --
  ```

**3. 결과 출력**

- 최종 우승자(가장 멀리 이동한 자동차)를 '`최종 우승자 : pobi`' 형태로 출력한다.
- 공동 우승자가 있을 경우 쉼표(,)로 구분한다.

**4. 예외 처리**

- 사용자가 잘못된 값을 입력할 경우 `Error`를 발생시키고 프로그램을 종료한다.
- `[ERROR]`로 시작하는 에러 메시지를 출력한다.
- 필요한 예외처리가 더 있을 경우 추가할 예정
